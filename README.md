A 2D Bullet Hell Shooter built from scratch in C++ using Visual Studio, featuring core gameplay systems, custom rendering, and full game flow management.  

Features

- Main Menu System Navigate between start, quit, and restart options  
- Player Controls Smooth 2D movement and shooting mechanics  
- Enemy System AI-based enemies with shooting patterns and spawn management  
- Projectile System Separate bullet pools for player and enemies  
- Level Scrolling Continuous vertical scrolling for the level background  
- Score System Points awarded for defeating enemies
- Pickup System Chance-based drops when defeating enemies:
  - Health Pickup Restores player HP  
  - Weapon Upgrade Pickup Temporarily changes the player firing mode      
- Collision System Player/enemy and bullet collision detection  
- HUD System Displays player lives, score  
- Audio System  Background music  
- Wave Spawner with increasing difficulty
